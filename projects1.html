Projects

Pacman Assignment

A web page that produces a PacMan image with the click of the button, that move across the page at various speeds and directions. The main learning outcomes addressed with this exercise were learning to use JavaScript to dynamically display data on a web page, manipulating the DOM, and using array methods. Skills that were learned in this project were looping over an array and moving images to the DOM, detecting collision within the window, creating a factory to make new objects with random positions and velocities, and scaling the random values from the Math() function.

Eye Movement Exercise

A web page created with two eyes that follow the user’s mouse movement. The exercise was geared to require integrating skills learned in JavaScript and CSS to allow the eyes to appear and follow the dynamic movements of the user’s mouse. The exercise also was an introduction to how mouse events are a popular way to link the DOM elements to events. Starter code was given and the task was to add a second eye and have the eye follow the mouse movements as well.

Real Time Bus Tracker

A coding assignment dealing with HTML, JavaScript, and CSS languages, while also using Mapbox library for mapping purposes. I used a map of the campuses of MIT and Harvard and adding in an animated marker on the map to highlight the bus route by using the MTBA bus data to determine exactly where the stops occur. To complete this tracker we had use access tokens from Mapbox to render maps on the webpage, and then create a marker using the pre-built Mapbox global functions. This task had the learning outcomes of rendering a map on a webpage and learning different ways to manipulate the map.
